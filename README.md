# Laravel Design Pagination

Pagination component for Laravel. Supports Livewire, Blade, and Vue 3.

## Installation

```bash
composer require mrshanebarron/pagination
```

## Usage

### Livewire Component

```blade
<livewire:ld-pagination />
```

### Blade Component

```blade
<x-ld-pagination />
```

## Configuration

Publish the config file:

```bash
php artisan vendor:publish --tag=ld-pagination-config
```

## Customization

### Publishing Views

```bash
php artisan vendor:publish --tag=ld-pagination-views
```
